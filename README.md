# IOT-based-air-guard-system

##  Project Title
IoT-Based Air Guard System (Fire & Gas Leak Detection)

##  Description
This project is an IoT-based safety system using ESP32 to detect fire and gas leakage in real-time and trigger instant alerts using buzzer and LEDs.

##  Components Used
- ESP32 Microcontroller  
- MQ2 Gas Sensor  
- Temperature Sensor (LM35 / DHT11)  
- Buzzer  
- Red LED (Danger Indicator)  
- Green LED (Safe Indicator)  

##  Working Principle
- Sensors continuously monitor air conditions  
- MQ2 detects gas/smoke leakage  
- Temperature sensor detects fire conditions  
- If threshold is exceeded → buzzer ON + Red LED ON  
- Otherwise → Green LED ON  

##  Features
- Real-time monitoring  
- Instant alert system  
- Low-cost and efficient  
- Easy implementation  

##  Future Scope
- Mobile app integration (Blynk/Firebase)  
- Cloud data monitoring  
- AI-based fire prediction  
- Smart home integration
